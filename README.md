# UTC-Link — Plataforma Inteligente de Inserción Laboral

UTC-Link es una plataforma creada durante el Hackathon de la Universidad Tecnológica de Coahuila (UTC) que impulsa la vinculación entre estudiantes y egresados con el sector productivo.

A través de tecnologías de inteligencia artificial, web scraping y machine learning, centraliza vacantes, optimiza currículums y prepara a los usuarios para entrevistas, facilitando su transición al mundo laboral.

---

## Problema

Los estudiantes y egresados de la UTC suelen enfrentarse a:
- Vacantes dispersas en múltiples plataformas.
- Falta de retroalimentación objetiva para mejorar su CV.
- Escasa preparación para realizar entrevistas profesionales.

---

## Solución

UTC-Link proporciona:
- Centralización de vacantes mediante scraping y clasificación inteligente.
- Revisión automatizada de CV con recomendaciones de mejora.
- Simulador de entrevistas con retroalimentación para entrenar habilidades.

La plataforma actúa como un asistente integral que ayuda a mejorar las oportunidades de inserción laboral.

---

## Características principales

1. Buscador inteligente de vacantes  
   - Recomendaciones según perfil  
   - Clasificación automática  

2. Análisis y optimización de CV  
   - Identificación de áreas de mejora  
   - Sugerencias basadas en palabras clave  

3. Simulación de entrevistas  
   - Preguntas adaptadas al perfil  
   - Retroalimentación automática  

---

## Tecnologías

| Capa | Tecnologías |
|------|-------------|
| Frontend | Next.js |
| Backend | FastApi |
| IA | OpenAI / NLP / ML |
| Scraping | Python / Puppeteer / BeautifulSoup |
| Base de datos | PostgreSQL |
| Infraestructura | Docker / Vercel / Render |

*Las tecnologías están sujetas a cambios durante el desarrollo.*
